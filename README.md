# Halsoprofilen-Client-Administration-System

Administration System for managing gym membership and clients

![logo](https://github.com/user-attachments/assets/6e28b087-393c-45c8-b2b7-c28b58f9f802)

### Create docker image:

```
docker build -t halsoprofilen .
```

### Create docker container:

```
 docker stop halsoprofilen-app; docker rm halsoprofilen-app; docker-compose up --build -d
```
